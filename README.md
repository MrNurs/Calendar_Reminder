# Calendar_Reminder
Reminder Calendar App
A simple but structured calendar-based application built with modern Android compose and articture technologies.

The main idea of this project is to demonstrate how a seemingly simple task management concept can become clean, scalable, and maintainable when proper architecture and state management are applied.
